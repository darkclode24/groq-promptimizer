# Groq API Key Meta Prompting Tool

*Generate and manage meta prompts using the Groq API from your terminal.*

## Description

The **Groq API Key Meta Prompting Tool** is a lightweight CLI utility for creating, refining, and managing meta prompts powered by the Groq API. It’s designed for developers who want fast, scriptable prompt workflows without building a full interface.

## Prerequisites

* Python ≥ 3.9
* `pip` or `poetry`
* A valid Groq API key

## Installation

Clone the repository and install dependencies.
